minitrade a minimalist trade engine


Requirements
============
The *minitrade* library has the following dependencies:

- Python 3.8
- Sphinx 4.0.2
- pytest 6.1.1
- pytest-cov 2.12.1
- tox 3.23.1
- pypi

Recommendation
============
- Annaconda 3.8

Installation
============

To download the mini-trade library from Github, do::

    $ git clone https://github.com/janFiOpelito/minitrade.git
